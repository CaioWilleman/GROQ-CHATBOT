# 🤖 GROQ Chat

Chatbot conversacional com IA desenvolvido em Python.

## Funcionalidades

- Conversa em linguagem natural
- Memória da conversa (lembra o que foi dito antes)
- Personalidade customizável via system prompt
- Variáveis de ambiente com .env

## Tecnologias

- Python
- Groq API (LLaMA 3.3)
- python-dotenv

## Como rodar

1. Clone o repositório
2. Instale as dependências:
   pip install groq python-dotenv
3. Crie um arquivo .env com sua chave:
   GROQ_API_KEY=sua_chave_aqui
4. Rode:
   python app.py
